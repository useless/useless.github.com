Flashless

Ein WebKit-Plugin, das eine Vorschau für Flash-Video-Dienste anzeigt und den Rest blockiert.

Herunterladen (.dmg, ≈380K)

Universal (32/64bit)

Voraussetzungen

Mac OS X 10.5 Leopard oder
10.6 Snow Leopard und
ein WebKit-Browser (z.B. Safari).

26Dez2010 — Versionshinweise.

Anleitung, Quellen

  • Flash blockieren

    Flashless verhindert das automatische Laden von Flash-Inhalten. Details

  • Video-Vorschaubilder

    So bekommen Sie einen schnellen Überblick über die Videos. Details

  • Ohne Flash abspielen HD

    Sehen Sie unterstützte Videos mit QuickTime an. Mit Safari 5 sogar im Vollbild. Details

  • Videos herunterladen HD

    Laden Sie Videos von bekannten Videodiensten auf ihren Rechner. Jetzt auch hochaufgelöste Videos. Details

  • Werbung entfernen

    Werden Sie Anzeigen mit einem einfachen Menübefehl los. Details

  • Zum Original gehen

    Sie können eingebettete Videos auf der Seite des Anbieters sehen. Details
