Flashless

Ein WebKit-Plugin, das eine Vorschau für Flash-Video-Dienste anzeigt und den Rest blockiert.

Voraussetzungen

Mac OS X 10.5 Leopard oder
10.6 Snow Leopard und
ein WebKit-Browser (z.B. Safari).

Anleitung, Quellen

  • Flash blockieren — Flashless verhindert das automatische Laden von Flash-Inhalten. Details …
  • Video-Vorschaubilder ermöglichen einen schnellen Überblick über die Videos. Details …
  • Ohne Flash abspielenVerbessert — sehen Sie unterstützte Videos standardmäßig mit Quicktime an. Details …
  • Videos herunterladen von vielen bekannten Videodiensten. Details …
  • Werbung entfernen mit einem einfachen Menübefehl. Details …
  • Zum Original gehen, um Videos auf der Seite des Anbieters zu sehen. Details …
