Flashless

A WebKit plug-in to display preview images for Flash based video services and block the rest.

Download

Version 1.4, requires Mac OS X 10.5.

Unzip, place the .webplugin-package in “~/Library/Internet Plugins/” and restart Safari.

Features

Supports the following video services:

  • Flickr — preview
  • Google Video — preview, download
  • Viddler — preview
  • Vimeo — preview
  • XTube — preview, download
  • YouTube — preview, download (only on youtube.com)

Allows you to show or remove all elements from a source until you restart Safari. NEW

Sources
