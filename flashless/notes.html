Flashless

A WebKit plug-in to display preview images for Flash based video services and block the rest.

Version 2.0.2

“The Birds” — 2Dec2009

  • Increased reliability of the YouTube, Flickr and Blip.tv services.
  • Reenabled downloads from TwitVid.

Version 2.0.1

“Panic” — 6Nov2009

  • Fix YouTube downloads and TwitVid previews.

Version 2.0

“Delegation” — 20Oct2009

  • Increased performance thanks to multithreading.
  • Display Icons also in very small Flash elements.
  • Use command/option/shift keys for original/download/direct play.
  • Support for Blip.tv, Vimeo, Flickr, and universal YouTube downloads.

Version 1.5

“Clusterimage” — 09Sep2009

  • Mac OS X 10.6 Snow Leopard and 64 bit are fully supported.
  • Icon changes when pressing option key where download is available.
  • Open original for a video.
